# User Stories

Working draft for the first pass of product scope.

## Core wallet

- As a new user, I want to create a wallet and back it up safely.
- As an existing Solana user, I want to import a wallet from a seed phrase or private key.
- As a returning user, I want to unlock quickly with biometrics or a password.

## Money movement

- As a wallet user, I want to send SOL or SPL tokens without guessing the real fee impact.
- As a wallet user, I want Helio to warn me before a send would leave the account in a bad state.
- As a wallet user, I want to receive funds with a QR code and a copyable address.

## Active usage

- As a trader, I want to swap tokens without leaving the wallet.
- As a staker, I want to compare validators and manage stake in one place.
- As a DeFi user, I want to review dApp requests in plain language before signing.
